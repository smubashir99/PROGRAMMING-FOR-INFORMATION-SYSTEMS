# Sum of Multiples

## Description
This project is a simple JavaScript assignment that calculates the sum of elements in one array that are multiples of elements in another array.

## Functionality
- Takes two arrays `f` and `m` from user input
- Uses `split()` and `parseInt()` to process input
- Finds values in `m` that are divisible by values in `f`
- Displays the final sum in `<div id="output">`
